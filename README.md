# Electronic-Shop
This project is a theme - online store. 

Language: Java<br/>
Frameworks: Spring Core, Spring MVC, Spring Security.<br />
Database: MySQL, Hibernate.<br/>
Front-end: HTML, CSS, AngularJS.<br/>
Tools: Maven, IntelliJ IDEA, Git, Lombok.<br/>
